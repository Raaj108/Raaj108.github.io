  2. Saves the user data in MonboDB (password is encrypted before storing).

RWeatherApp

Get accurate, local and detailed weather forecast.
Previous Next
Overview

RWeatherApp is a simple single page Progressive Web Application(PWA) that provides accurate and detailed weather forecast, available at any location or city in the world. The RWeatherApp is built with PWA methodology using AngularJS, JavaScript, HTML5, CSS3 and Bootstrap to deliver fast, engaging, and reliable mobile web experiences even when the user is offline or on an unreliable network.

  1. Accurate, local and detailed weather data.
  2. 5 day forecast with weather data for every 3 hours.
  3. Search weather information by city name.
  4. Graphical display of temperatures.
